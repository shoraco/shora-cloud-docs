# Welcome to Shora Cloud

Shora Cloud is a headless, API-first commerce platform that enables developers to build modern e-commerce experiences. Our platform provides everything you need to create, manage, and scale your e-commerce applications.

Build your frontend with any technology stack using our powerful APIs:

```typescript
import { ShoraCloudClient } from '@shoracloud/sdk';

const client = new ShoraCloudClient({
  apiKey: process.env.SHORA_API_KEY,
});

// Fetch products
const products = await client.products.list({
  limit: 10,
  category: 'electronics',
});
```
